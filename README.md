## [McMod Server](mcmod-server)
[![Version](https://img.shields.io/badge/Latest-2.0.0-blue)](#)
[![python3]](#)

*(McDonald's only)*

This server is needed to generate coupons for the Telegram Bot and to use the
McMod app.
